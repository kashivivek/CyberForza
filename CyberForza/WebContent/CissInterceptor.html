CISS INTERCEPTOR

CISS Interceptor provides granular visibiltity into the network and infrastructure allowing IT and network administrators to see who is accessing data, Internet sites and applications that cause increased risks. Traditional existing solutions are bound to the TCP/IP protocol stack and only recognize IP addresses of devices on the network, not the actual user.

Threats have become more sophisticated, internal users can also carry out attacks, so monitoring the internal risks is a key requirement creating a need for an identity-based solution that maps the IP address to the user name or user group.

Cyber Forza Interceptor provides the ability to either allow or deny access to files, Internet sites and applications can be based on a user's access rights, determined by the user's or the user group's business needs.

CISS Interceptor Benefits

  • CISS Interceptor (CISS Defender + IAM) is an integrated solution of CISS Defender and Active Directory / LDAP server to provide threat centric identity and access security to track internal and external threats.
  • CISS Interceptor provides an integrated network and infrastructure solution as opposed to an IAM infrastructure only product.
  • As businesses migrate to cloud computing and the economies of scale it has to offer, CISS Interceptor provides the enterprises with a Threat Centric IAM approach.
